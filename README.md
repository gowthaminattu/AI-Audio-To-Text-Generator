#  AI Audio to Text Generator

**A web-based application that converts speech/audio into text using Speech Recognition and basic NLP processing.**

---

##  Features
-  Record audio directly
-  Upload audio files (.wav)
-  Instant speech → text conversion
-  Download output as text/PDF
-  Clean and responsive UI

##  Installation (Run Locally)

1. Clone the repository
 git clone https://github.com/gowthaminattu/AI-Audio-To-Text-Generator.git
cd AI-Audio-To-Text-Generator

2. Create virtual environment
 python -m venv venv
 venv\Scripts\activate

3. Install dependencies
   pip install -r requirements.txt
   
4. Run the app
   python app.py
   
5. Open in browser
   http://127.0.0.1:5000
   
---

##  How it Works

1. User uploads or records audio  
2. Audio is processed using Speech Recognition  
3. Text output is generated  
4. Optional NLP improves formatting
